Project
Obstacle Avoiding Moving Car

The Obstacle Avoiding Moving Car is a microcontroller-based robotics project developed during my college coursework. The system uses an ultrasonic sensor to continuously measure the distance between the vehicle and nearby objects. When an obstacle is detected within a predefined range, the controller processes the data and automatically changes the motor direction to avoid collisions.

  • Microcontroller
  • Ultrasonic Sensor
  • Motor Driver
  • Power Supply

This project provided hands-on experience in embedded systems, sensor integration, and motor control. It enhanced my problem-solving skills related to hardware debugging and real-time decision making. The project demonstrates the core concepts of autonomous navigation and serves as a foundation for advanced robotics and automation systems.

Bitcoin Price Tracker

This project is a web-based application focused on Bitcoin that allows users to view real-time Bitcoin prices, market trends, and basic cryptocurrency information. The application fetches live data from a public API and displays it in a clean, responsive, and user-friendly interface. This project helped in understanding API integration and dynamic data handling in web development.

  • Bootstrap 5
  • JavaScript
  • Flexbox
  • Git & GitHub
